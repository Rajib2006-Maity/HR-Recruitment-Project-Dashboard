# HR-Recruitment-Project-Dashboard

A modern, responsive web application for managing the recruitment process. This project provides an intuitive dashboard to track candidates, job openings, interview schedules, and hiring progress through a clean and user-friendly interface.

## 🚀 Features

- 📋 Recruitment management dashboard
- 👥 Candidate tracking
- 💼 Job opening management
- 📅 Interview scheduling
- 📊 Hiring progress overview
- 🎨 Modern responsive UI
- ⚡ Fast and lightweight
- 📱 Mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Tabler Icons

## 📂 Project Structure

```
recruitment-project/
│
├── index.html
└── README.md
```

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/recruitment-project.git
```

2. Navigate to the project folder

```bash
cd recruitment-project
```

3. Open `index.html` in your browser.

No additional dependencies or installation are required.

## ▶️ Usage

- Launch the application by opening `index.html`.
- Browse the recruitment dashboard.
- Manage candidates and recruitment workflow through the interface.
- View recruitment metrics and hiring status.
